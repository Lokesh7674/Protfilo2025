My Projects

Smart Bridge Automatic Height Adjustment

An innovative bridge system that automatically adjusts its height based on water levels using Arduino, servo motors, and moisture sensors.

This system enhances safety during floods and heavy rainfall by preventing bridge collapse and traffic disruption.

Wireless Discharge System

A contactless energy discharge system using inductive coupling to transfer energy without physical connections.

This technology eliminates spark risks, reduces wear and tear, and is ideal for EV batteries, medical devices, and defense applications.

Order Control Management System

A Python-based application for managing customer orders, inventory tracking, and sales operations.

Features database integration, user authentication, and real-time order tracking with a focus on clean code and scalability.
